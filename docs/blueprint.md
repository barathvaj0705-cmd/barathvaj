# **App Name**: StudyBuddy AI

## Core Features:

- Secure Sign-Up/Login: Allows students to create accounts and log in securely to track their progress and personalize their learning experience.
- AI Chatbot Tutor: An AI chatbot powered by LLMs, which provides instant study help by answering questions, explaining concepts, and offering guidance.
- Customizable Quiz Generator: Enables students to generate quizzes based on selected topics and difficulty levels to test their understanding.
- Quiz History: Allows students to review past quiz results and identify areas for improvement.
- Chatbot Interaction History: Maintained history of questions and answers for personalized revision using AI tool. LLM tool identifies opportunities to improve students performance, based on historical data and quiz performance.

## Style Guidelines:

- Primary color: Vibrant blue (#4285F4), evoking trust, intelligence, and a modern feel.
- Background color: Clean white (#F9FBFF), creating a sense of clarity and focus.
- Accent color: Light blue (#74A7F8) to complement the primary and background, for a bright and airy feel.
- Body and headline font: 'Inter' for a clean, modern, and highly readable user experience. 
- Use a set of consistent icons in a line-art style, filled in with the primary blue color to represent various subjects, quizzes, and chatbot functions.
- Maintain a clean and intuitive layout with clear information hierarchy, making it easy for students to navigate and access resources.
- Subtle animations for feedback and state transitions
